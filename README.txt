This repository contains key python scripts to reproduce results of a paper entitled "Wildfire synchronicity across European regions favored by specific weather regimes" (doi) by Xinhang Li, Raul R. Wood, Julia Miller, Dominik Büeler, and Manuela I. Brunner. Besides standard libraries of python, you may need to install the following python libraries to run these scripts:

-pandas (Version: 2.2.2)
-geopandas (Version: 0.14.3)
-xarray (Version: 2024.3.0)
-numpy (Version: 1.26.4)
-rioxarray (Version: 0.15.3)
-shapely (Version: 2.0.3)
-regionmask (Version: 0.12.1)
-cartopy (Version: 0.23.0)
-scipy (Version: 1.13.0)

Please also note that some scripts were run on HPC.

All input data are publicly available as described in the 'Data and code availability' statement in the paper.
